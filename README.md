# TCC
Material utilizado e desenvolvido no Trabalho de Conclusão de Curso “Análise de Sensibilidade da Política de Estoques (s, S) via Simulação de Monte Carlo”. O objetivo do projeto é avaliar como os parâmetros fundamentais de um sistema de reposição, como demanda, custos e lead time influenciam os níveis ótimos de estoque e o custo total médio
